## 一、摸底结论：当前系统任务可视化现状

基于 t1 竞品调研（五大框架四大特征对比）和现有源码审计，AgentOS 在可观测性维度存在严重短板：

| 维度 | 现状 | t2 盲区报告评级 |
|------|------|----------------|
| 任务执行追踪 | 仅 Task 级别的 status + started_at/completed_at | 🔴 严重缺失 |
| 节点/步骤粒度 | 零——orchestrator 的 PlanStep 有 duration 但未持久化 | 🔴 完全缺失 |
| 时间线可视化 | 无——前端 Tasks 页仅表格+模态弹窗，无甘特/瀑布流 | 🔴 完全缺失 |
| 执行日志 | ExecutionLog 表有 duration_ms，但未关联步骤 | 🟡 有数据，缺关联 |
| 竞品参考 | ChatDev 2.0 WebSocket 可视化、LangGraph SSE stream_mode | ✅ 竞品已有 |

**关键发现（盲区报告 t2）：** t2 报告将可观测性评为 "print+SSE" 即一笔带过，实际情况更糟——无结构化日志、无 Metrics、无告警、无健康检查端点。

```python
class TaskExecutionNode(models.Model):
    """任务执行节点 —— 描述 Task 内部的步骤/子任务执行拓扑"""

    class Status(models.TextChoices):
        PENDING = 'pending', '等待中'
        RUNNING = 'running', '执行中'
        DONE = 'done', '已完成'
        FAILED = 'failed', '失败'
        SKIPPED = 'skipped', '已跳过'
        TIMED_OUT = 'timed_out', '超时'

    # ── 拓扑 ──
    task = models.ForeignKey(
        Task, on_delete=models.CASCADE,
        related_name='execution_nodes',
    )
    node_key = models.CharField(
        max_length=32,
        help_text='节点唯一标识（同 task 内唯一，如 t1, t2, step_01）',
    )
    parent_node = models.ForeignKey(
        'self', on_delete=models.CASCADE,
        null=True, blank=True,
        related_name='child_nodes',
        help_text='父节点（支持嵌套子步骤）',
    )
    depends_on = models.JSONField(
        default=list, blank=True,
        help_text='依赖的 node_key 列表 ["t1", "t2"]',
    )
    level = models.PositiveSmallIntegerField(
        default=0, db_index=True,
        help_text='节点层级（0=根节点，1/2/3=子节点）',
    )

    # ── 执行信息 ──
    agent_name = models.CharField(
        max_length=64, blank=True,
        help_text='执行 Agent（banni | basir | 豆角云枢）',
    )
    action = models.CharField(
        max_length=32,
        help_text='节点动作类型: plan | spawn | search | terminal | read_file | write_file | reason | reflect | reply',
    )
    description = models.TextField(
        blank=True,
        help_text='节点描述（如 "搜索 Leader & Workers 定义"）',
    )

    # ── 状态与结果 ──
    status = models.CharField(
        max_length=16, choices=Status.choices,
        default=Status.PENDING, db_index=True,
    )
    result = models.TextField(blank=True)
    error_info = models.JSONField(default=dict, blank=True)

    # ── 时间轴（核心：支持甘特/瀑布流） ──
    started_at = models.DateTimeField(null=True, blank=True)
    completed_at = models.DateTimeField(null=True, blank=True)
    duration_ms = models.PositiveIntegerField(
        default=0,
        help_text='实际执行耗时（毫秒）',
    )
    estimated_duration_ms = models.PositiveIntegerField(
        default=0,
        help_text='预估耗时（用于甘特条灰色部分）',
    )

    # ── 统计 ──
    retry_count = models.PositiveSmallIntegerField(default=0)
    llm_calls = models.PositiveSmallIntegerField(
        default=0,
        help_text='本节点触发的 LLM 调用次数',
    )
    token_used = models.PositiveIntegerField(
        default=0,
        help_text='本节点消耗的 token 数（估算）',
    )

    # ── 关联 ──
    execution_log = models.ForeignKey(
        ExecutionLog, on_delete=models.SET_NULL,
        null=True, blank=True,
        related_name='execution_nodes',
        help_text='关联的详细执行日志',
    )

    created_at = models.DateTimeField(auto_now_add=True)
    updated_at = models.DateTimeField(auto_now=True)

    class Meta:
        db_table = 'task_execution_nodes'
        verbose_name = '任务执行节点'
        verbose_name_plural = verbose_name
        ordering = ['task', 'level', 'started_at']
        indexes = [
            models.Index(fields=['task', 'node_key']),
            models.Index(fields=['task', 'status']),
            models.Index(fields=['task', 'level', 'started_at']),
        ]
        constraints = [
            models.UniqueConstraint(
                fields=['task', 'node_key'],
                name='uq_task_node_key',
            ),
        ]

    def __str__(self):
        return f'[{self.node_key}] {self.description[:40]} ({self.get_status_display()})'

    @property
    def duration_seconds(self) -> float:
        return self.duration_ms / 1000.0

    @property
    def is_terminal(self) -> bool:
        """是否为叶子节点（无子节点）"""
        return not self.child_nodes.exists()
```

### 2.3 数据填充来源映射

| 数据来源 | 对应字段 | 填充时机 |
|---------|---------|---------|
| Yunshu PlanGraph | node_key, description, depends_on, level | PLAN 命令解析后 |
| orchestrator PlanStep | action, agent_name, node_key | PlanStep 创建时 |
| yunshu_io SPAWN_BANNI | agent_name, description, parent_node | 子进程 spawn 时 |
| yunshu_io WAIT_ALL + CHECK | status, result | 子进程完成/失败时 |
| orchestrator _execute_step | started_at, completed_at, duration_ms | 步骤开始/结束时 |
| REFLECT 自检 | action="reflect", description | ReflectState 执行时 |

**写入时机设计：**

```
1. Yunshu 输出 PLAN → 批量创建 TaskExecutionNode（status=pending）
2. SPAWN_BANNI → 更新对应节点 status=running, started_at=now
3. 子进程 stdout 捕获 → 更新 result
4. CHECK 完成/DONE → 更新 status=done/failed, completed_at=now, duration_ms
5. orchestrator 执行步骤 → 创建/更新节点，记录 started_at 和 duration_ms
6. REFLECT 阶段 → 创建节点 action="reflect"，记录评估结果
```

```python
# agents/views.py TaskViewSet 扩展

@action(detail=True, methods=['get'], url_path='nodes')
def get_node_tree(self, request, pk=None):
    """获取任务完整节点树 + 时间线"""
    task = self.get_object()
    root_nodes = task.execution_nodes.filter(
        parent_node__isnull=True
    ).select_related('execution_log').annotate(
        log_count=Count('detailed_logs'),
    ).order_by('started_at')

    nodes_data = TaskNodeSerializer(root_nodes, many=True).data

    # 计算汇总
    all_nodes = task.execution_nodes.all()
    total_duration = sum(n.duration_ms for n in all_nodes)
    total_llm = sum(n.llm_calls for n in all_nodes)
    total_tokens = sum(n.token_used for n in all_nodes)

    # 生成时间线（扁平化，适配甘特图）
    task_start = task.started_at
    timeline = []
    for n in all_nodes.order_by('started_at'):
        start_offset = 0
        if task_start and n.started_at:
            start_offset = int((n.started_at - task_start).total_seconds() * 1000)
        end_offset = start_offset + n.duration_ms

        # Gantt color by status
        color_map = {
            'done': '#1D9E75', 'running': '#378ADD',
            'failed': '#E24B4A', 'pending': '#6B7280',
            'skipped': '#9CA3AF', 'timed_out': '#F59E0B',
        }

        timeline.append({
            'node_key': n.node_key,
            'agent_name': n.agent_name,
            'action': n.action,
            'description': n.description,
            'status': n.status,
            'level': n.level,
            'depends_on': n.depends_on,
            'start_ms': start_offset,
            'end_ms': end_offset,
            'duration_ms': n.duration_ms,
            'color': color_map.get(n.status, '#6B7280'),
        })

    return Response({
        'task_id': task.id,
        'task_title': task.title,
        'task_status': task.status,
        'total_nodes': len(all_nodes),
        'total_duration_ms': total_duration,
        'total_llm_calls': total_llm,
        'total_tokens': total_tokens,
        'nodes': nodes_data,
        'timeline': timeline,
    })


@action(detail=True, methods=['get'], url_path='timeline')
def get_timeline(self, request, pk=None):
    """获取任务时间线（纯甘特图数据，比 /nodes/ 更轻量）"""
    task = self.get_object()
    task_start = task.started_at
    nodes = task.execution_nodes.order_by('started_at')

    color_map = {
        'done': '#1D9E75', 'running': '#378ADD',
        'failed': '#E24B4A', 'pending': '#6B7280',
        'skipped': '#9CA3AF', 'timed_out': '#F59E0B',
    }

    events = []
    for n in nodes:
        start_offset = 0
        if task_start and n.started_at:
            start_offset = max(0, int(
                (n.started_at - task_start).total_seconds() * 1000
            ))
        events.append({
            'node_key': n.node_key,
            'agent_name': n.agent_name,
            'action': n.action,
            'description': n.description,
            'status': n.status,
            'level': n.level,
            'depends_on': n.depends_on,
            'start_ms': start_offset,
            'end_ms': start_offset + n.duration_ms,
            'duration_ms': n.duration_ms,
            'color': color_map.get(n.status, '#6B7280'),
        })
    return Response({
        'task_id': task.id,
        'task_started_at': task_start,
        'task_completed_at': task.completed_at,
        'total_duration_ms': (
            int((task.completed_at - task_start).total_seconds() * 1000)
            if task_start and task.completed_at else 0
        ),
        'events': events,
    })
```

```python
# 位置1：PLAN 命令解析后 → 批量创建节点
# 在 YunshuCommandHandler._handle_plan() 中：
def _persist_plan_nodes(self, plan_graph: PlanGraph):
    """将 PlanGraph 的节点持久化为 TaskExecutionNode"""
    task = Task.objects.get(id=self.parent_id)
    for node in plan_graph.nodes:
        TaskExecutionNode.objects.update_or_create(
            task=task,
            node_key=node.task_id,
            defaults={
                'agent_name': node.agent_type,
                'action': 'spawn',
                'description': node.description,
                'depends_on': node.dependencies,
                'status': TaskExecutionNode.Status.PENDING,
                'level': 0,
            },
        )

# 位置2：SPAWN_BANNI / SPAWN_BASIR 执行时 → 更新节点状态
# 在 _spawn_child() 中：
def _update_node_running(self, spawn_id: str):
    """子进程启动时更新节点"""
    TaskExecutionNode.objects.filter(
        task_id=self.parent_id,
        node_key=spawn_id,
    ).update(
        status=TaskExecutionNode.Status.RUNNING,
        started_at=timezone.now(),
    )

# 位置3：子进程完成/WATCH → 更新节点结果
def _update_node_done(self, child_id: str, result: str, status: str):
    """子进程完成时更新节点"""
    node = TaskExecutionNode.objects.filter(
        task_id=self.parent_id,
        node_key=child_id,
    ).first()
    if node:
        now = timezone.now()
        duration = int((now - node.started_at).total_seconds() * 1000) if node.started_at else 0
        node.status = TaskExecutionNode.Status.DONE if status == 'DONE' else TaskExecutionNode.Status.FAILED
        node.result = result[:5000] if result else ''
        node.completed_at = now
        node.duration_ms = duration
        node.save()

# 位置4：REFLECT 阶段 → 创建评估节点
def _create_reflect_node(self, passed: bool, reason: str = ''):
    TaskExecutionNode.objects.create(
        task_id=self.parent_id,
        node_key=f'reflect_r{self.reflect_state.current_round}',
        action='reflect',
        description=f'REFLECT 第{self.reflect_state.current_round}轮自检',
        status=TaskExecutionNode.Status.DONE if passed else TaskExecutionNode.Status.FAILED,
        result=f'{"✅ 通过" if passed else "❌ " + reason}',
        level=0,
        started_at=timezone.now(),
        completed_at=timezone.now(),
    )
```

### 4.2 orchestrator.py 织入点

```python
# orchestrator.py 关键织入位置

# 在 _execute_step() 中：
def _execute_step_with_node(step: PlanStep, results: dict, task_id: int) -> tuple:
    """带节点持久化的步骤执行"""
    now = timezone.now()

    node, _ = TaskExecutionNode.objects.get_or_create(
        task_id=task_id,
        node_key=step.id,
        defaults={
            'agent_name': step.agent,
            'action': step.action,
            'description': step.note or step.command[:100],
            'depends_on': step.depends_on,
            'level': 0,
        },
    )

    # 标记开始
    node.status = TaskExecutionNode.Status.RUNNING
    node.started_at = now
    node.save()

    # 执行步骤
    t0 = time.time()
    status, result = _execute_step(step, results)
    elapsed_ms = int((time.time() - t0) * 1000)

    # 更新结果
    node.status = (
        TaskExecutionNode.Status.DONE if status == 'done' else
        TaskExecutionNode.Status.FAILED if status == 'failed' else
        TaskExecutionNode.Status.SKIPPED
    )
    node.result = result[:5000]
    node.completed_at = timezone.now()
    node.duration_ms = elapsed_ms
    node.llm_calls = step.llm_calls
    node.retry_count = step.retry_count
    node.save()

    return status, result
```

```jsx
  if (!data || !data.nodes || data.nodes.length === 0)
    return <div className="timeline-empty">该任务暂无执行节点数据</div>;

  return (
    <div className="task-timeline">
      {/* 汇总统计 */}
      <div className="timeline-header">
        <span>🔢 {data.total_nodes} 个节点</span>
        <span>⏱ {formatDuration(data.total_duration_ms)}</span>
        <span>🤖 {data.total_llm_calls} 次 LLM 调用</span>
        <span>💰 {data.total_tokens} tokens</span>
      </div>

      {/* 双栏布局 */}
      <div className="timeline-panel">
        <div className="timeline-tree">
          <NodeTree nodes={data.nodes} />
        </div>
        <div className="timeline-gantt">
          <GanttChart events={data.timeline} totalMs={data.total_duration_ms} />
        </div>
      </div>

      {/* 图例 */}
      <div className="timeline-legend">
        <LegendItem color="#1D9E75" label="已完成" />
        <LegendItem color="#378ADD" label="执行中" />
        <LegendItem color="#E24B4A" label="失败" />
        <LegendItem color="#6B7280" label="等待中" />
        <LegendItem color="#F59E0B" label="超时" />
        <LegendItem color="#9CA3AF" label="已跳过" />
      </div>
    </div>
  );
}
```

### 5.5 甘特图组件 (GanttChart)

```jsx
function GanttChart({ events, totalMs }) {
  if (!events.length) return <div className="gantt-empty">无时间线数据</div>;

  // 计算甘特条位置（百分比定位）
  const maxEnd = Math.max(...events.map(e => e.end_ms), totalMs);
  const barHeight = 24;
  const rowHeight = 32;
  const labelWidth = 160;

  // 时间刻度
  const tickMs = maxEnd > 60000 ? 10000 : 5000; // >1min 用 10s 刻度
  const ticks = [];
  for (let t = 0; t <= maxEnd; t += tickMs) ticks.push(t);

  return (
    <div className="gantt-container">
      {/* 时间刻度头 */}
      <div className="gantt-header" style={{ paddingLeft: labelWidth }}>
        {ticks.map(t => (
          <div
            key={t}
            className="gantt-tick"
            style={{ left: `${(t / maxEnd) * 100}%` }}
          >
            {formatMsShort(t)}
          </div>
        ))}
      </div>

      {/* 甘特条 */}
      <div className="gantt-body">
        {events.map((evt, i) => (
          <div key={evt.node_key} className="gantt-row" style={{ top: i * rowHeight }}>
            {/* 行标签 */}
            <div className="gantt-label" style={{ width: labelWidth }}>
              <span className="gantt-agent-icon">
                {evt.agent_name === 'banni' ? '🐧' : evt.agent_name === 'basir' ? '📊' : '🎯'}
              </span>
              <span className="gantt-node-key">{evt.node_key}</span>
              <span className="gantt-action">{evt.action}</span>
            </div>

            {/* 甘特条 */}
            <div
              className="gantt-bar"
              style={{
                left: `${(evt.start_ms / maxEnd) * 100}%`,
                width: `${Math.max((evt.duration_ms / maxEnd) * 100, 0.5)}%`,
                backgroundColor: evt.color,
                marginLeft: labelWidth,
              }}
              title={`${evt.description}\n${formatDuration(evt.duration_ms)}`}
            >
              <span className="gantt-bar-text">
                {formatDuration(evt.duration_ms)}
              </span>
            </div>

            {/* 依赖线 */}
            {evt.depends_on?.map(depKey => {
              const dep = events.find(e => e.node_key === depKey);
              if (!dep || dep.end_ms > evt.start_ms) return null;
              return (
                <svg
                  key={`${depKey}-${evt.node_key}`}
                  className="gantt-dependency-line"
                  style={{ position: 'absolute', top: 0, left: 0, width: '100%', height: '100%' }}
                >
                  <line
                    x1={`${(dep.end_ms / maxEnd) * 100 + labelWidth / maxEnd * 100}%`}
                    y1={i * rowHeight + rowHeight / 2}
                    x2={`${(evt.start_ms / maxEnd) * 100 + labelWidth / maxEnd * 100}%`}
                    y2={i * rowHeight + rowHeight / 2}
                    stroke="#6366f1"
                    strokeWidth="1"
                    strokeDasharray="3,3"
                  />
                </svg>
              );
            })}
          </div>
        ))}
      </div>
    </div>
  );
}
```
