Core Features

四大核心特征

基于对 AutoGen、CrewAI、LangGraph、MetaGPT、ChatDev 2.0 的深度调研，AgentOS 在每个维度都具备差异化优势

⚡

超步并行调度

基于 PlanGraph 依赖图自动分组并行批次。借鉴 LangGraph BSP 超步模型，同批次独立任务并发执行，比传统串行编排快 3-5 倍。

🛡️

生产级熔断器

业界唯一。五维实时监控（心跳/内存/超时/停滞/进度），300s 子任务硬限 + 600s 编排总超时 + 3 轮自检 FORCE_PASS 熔断。AutoGen/CrewAI/LangGraph 均无此能力。

📡

SSE 实时推送

原生 Django SSE 端点，逐轮工具调用 → 蓝色确认胶囊 → 灰色进度节点 → 最终报告。对标 LangGraph stream_mode 分层设计，支持 Web + 飞书双端同步。

🧠

文本协议编排

独创 10 命令文本协议（PLAN→SPAWN→WAIT→REFLECT→REPLY），LLM 原生理解，无需 YAML 配置。比 CrewAI Flows 更灵活，比 LangGraph StateGraph 更高层，比 ChatDev 2.0 拖拽更智能。
